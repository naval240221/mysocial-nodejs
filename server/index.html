My Social

This provides you ability to perform these api-endpoints

  1. Register a User
  2. Login
  3. Get a user specific details (Posts, Todo)
  4. Get users (Provide pagination, query based (email, account creation timestamp, firstname, lastname))
  5. User can add a post
  6. User can delete/edit thier own posts
  7. User can view their as well as other users post (Provide all the details)
  8. Users can comment on their and other's posts
  9. User can delete their comment
  10. User can fetch all comments related to a post (By default 50 latest comments will be served)
  11. Get a post details (Include all the comments)
  12. User can add a todo
  13. User can edit/delete/mark as completed
  14. User can view all todo (By default pending, can apply completed filrer using query, supports pagination)
